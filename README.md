### Products I've Built

These are live, production applications serving real users.

**OneTapTools** — All-in-one mobile toolkit with 8 tools: QR scanner, background remover (on-device ML), document scanner, OCR, PDF editor, file converter, AR ruler, and clipboard manager.
> **Stack:** React Native, Expo, TypeScript, Spring Boot, Python ML Worker, Redis, PostgreSQL
>
> [![Website](https://img.shields.io/badge/Website-onetaptools.app-blue?style=flat-square)](https://onetaptools.app)

**RezervOn** — Online booking platform for beauty & wellness businesses. 68+ pages, 4 dashboards (Client, Business Admin, Staff, System Admin), real-time notifications via WebSocket.
> **Stack:** React, Capacitor (iOS/Android), Java 25, Spring Boot, PostgreSQL, WebSocket/STOMP
>
> [![Website](https://img.shields.io/badge/Website-rezervon.al-blue?style=flat-square)](https://rezervon.al) [![Demo](https://img.shields.io/badge/Demo-demo.rezervon.al-green?style=flat-square)](https://demo.rezervon.al)

**ARCH Design Platform** — Project management and design tool for architects and interior designers. Mood boards, FF&E schedules, product library, client portal, task management with Kanban.
> **Stack:** React, TypeScript, TailwindCSS, Capacitor, Spring Boot, PostgreSQL

**AI Agent Framework** — Multi-agent AI system with 25 specialized agents that orchestrate software development. Knowledge layer, self-improvement via critique logs, and a SaaS platform.
> **Stack:** Claude Code, Next.js, Python, FastAPI, Docker
